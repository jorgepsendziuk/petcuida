se as pessoaas cadastrarem seus pets, elas podem compartilhar com as clinicas caso permita

VINCULAR POR CPF os pets (opcional para o cadastrante fornecer cpf, mas se fornecer podera compartilhar os pets nas clinicas que pedirem permissao de acesso)
assim as clinicas podem digitar o cpf e pedir permissao para visualizar detalhes dos pets individualmente e adicionar informacoes tratamentos etc, 

enquanto o tutor nao liberar o acesso ao pet a clinica pode supor o nome adicionar detalhes tratamentos etc atualizacoes do pet mas nao sera adicionadas ate o tutor liberar o acesso ao pet.

é vantagem para as pessoas que:
-terao acesso a ficha do pet com atualizacoes que as clinicas colocam, 
-o cadastro super facilitado usando IA (tirar uma foto e supor informacoes para o cadastro do pet), 
-interface ultra simples e facil de usar 

é vantagem pras clinicas que sera gratuito o uso do sistema e podem aparecer num catalogo de clinicas para os usuarios
